# Ben's Goals Dashboard

A clean, modern goal tracking dashboard for Vercel deployment.

## Features
- Dark mode GitHub-style UI
- Goal cards with status, priority, and next actions
- Model routing reference (Brain + Muscles strategy)
- Health summary table
- Mobile-friendly responsive design

## Deploy to Vercel

1. Import this repo to Vercel
2. No build settings needed (static HTML)
3. Deploy!

## Local Development

Just open `index.html` in a browser.

## Data

Goals are currently hardcoded in the HTML. Future versions may pull from an API or JSON file.
